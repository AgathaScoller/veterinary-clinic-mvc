# Veterinary Clinic MVC

A simple ASP.NET Core MVC project developed for a veterinary clinic management system.

## Features

- MVC Architecture
- Customer Model
- Supplier Model
- Home Controller
- Razor Views
- Bootstrap Tables
- In-memory data (no database)

## Technologies

- ASP.NET Core MVC
- C#
- .NET 6
- Razor Pages
- Bootstrap

## Project Structure

```
Controllers/
Models/
Views/
wwwroot/
Program.cs
```
